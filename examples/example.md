# codeDown Example

This file exists so you can quickly test `code-down`.

## Markdown Features

### Code

```python
def hello(name: str) -> str:
    return f"Hello, {name}!"


print(hello("codeDown"))
```

### Table

| Command | What it does |
| --- | --- |
| `code-down example.md` | Convert to PDF |
| `code-down -w example.md` | Watch + rebuild |
| `code-down themes` | Pick a theme |

### Quote

> Tip: Use `code-down config set-theme` to make a theme the default.
